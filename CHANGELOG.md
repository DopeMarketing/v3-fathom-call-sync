# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [0.1.0] - 2026-03-27

### Added
- Initial project scaffold generated by Project Generator
- Complete Next.js 15 application structure with App Router
- Supabase database integration with 7 table schema:
  - `users` table for user accounts and profiles
  - `oauth_connections` table for stored OAuth tokens
  - `sync_configurations` table for user sync preferences
  - `fathom_transcripts` table for cached transcript metadata
  - `sync_jobs` table for background job tracking
  - `sync_job_items` table for individual upload tracking
  - `uploaded_files` table for successful upload records
- Full route structure with 9 pages:
  - `/` - Landing page with project overview
  - `/dashboard` - Main sync dashboard
  - `/connections` - OAuth connection management
  - `/history` - Sync history and logs
  - `/settings` - User preferences and configuration
  - `/auth/fathom/callback` - Fathom OAuth callback
  - `/auth/google/callback` - Google OAuth callback
  - `/api/sync/trigger` - Manual sync trigger endpoint
  - `/api/sync/status` - Sync status monitoring endpoint
- Integration stubs for external APIs:
  - Google Drive API wrapper with upload capabilities
  - Fathom API client for transcript fetching
  - Google Sheets API integration for metadata tracking
  - n8n workflow template generator
  - Pipedream automation connectors
- TypeScript type definitions for:
  - Database schema and relationships
  - Fathom API response structures
  - Google Drive API interfaces
  - Sync job and status types
- Tailwind CSS styling system with dashboard-focused design
- Authentication flow preparation with OAuth callback routes
- Server Actions setup for secure API operations
- Environment variable configuration template
- Database migration files and RLS policies
- Complete documentation suite:
  - README.md with setup instructions
  - CLAUDE.md development briefing
  - TECHNICAL_DEBT.md tracking file
  - ROADMAP.md feature planning
- Vercel deployment configuration
- ESLint and TypeScript strict mode configuration
