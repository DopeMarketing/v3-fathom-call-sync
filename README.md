# V3 Fathom Call Sync

## Overview
V3 Fathom Call Sync is an automation tool that connects your Fathom account to Google Drive, automatically syncing meeting transcripts to a designated folder. It provides a simple dashboard interface for managing the sync process and monitoring transfer status.

## Problem
Manual process of downloading Fathom transcripts and organizing them in Google Drive is time-consuming and error-prone, leading to inconsistent backup practices

## Target User
Personal user who regularly uses Fathom for meeting transcription and wants automated backup/organization in Google Drive

## Features
- OAuth authentication for Fathom and Google Drive APIs
- Bulk transcript export and upload functionality
- Sync status dashboard with transfer monitoring
- Manual sync trigger with on-demand execution
- Duplicate detection and smart handling system
